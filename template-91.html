Migration Recommendation

When it comes to grids, there are several versions available, with EFX Grid being the latest one. We highly recommend migrating to EFX Grid, whether you're currently using Emerald Grid or Atlas Blotter. This migration should seamlessly integrate with Element Framework v6 or v7 and ensure smooth functionality.

You can following this guild line

  1. Migrate from Emerald Grid to Atlast Blotter
  2. Migrate from Atlas Blotter to EFX Grid
