addData(data, formatopt)

Add new data rows to this grid. This method will be deprecated in favor of tr.CompositeGrid#getDataTable due to slow data parsing involved in this method.
The data will be added to the internal DataTable.
The structure of the data can be in several formats depending on the value of the 'format' parameter. See the example below
Parameters:
data
Object | Array.<(Object|Array)>
Data object to be added. The structure is depending on the other parameter 'format'
format
string
<optional>
Format of the data. Possible values are 'array', 'rows', 'columns'
See:
  • tr.DataTable#addRows
Example:
// Rows format (Recommended)
var data = [
  {col1: 0, col2: 12, col3: 33},
  {col1: 1, col2: 13, col3: 44},
  {col1: 2, col2: 15, col3: 55}
]
//Array format (Default). The order of data in each row has to match the 'fields' property of the 'dataModel'
var data = [
  [true, "row1", 0, 1, 2],
  [true, "row2", 0, 1, 2],
  [false, "row3", 0, 1, 2]
];

//Column format
var data = {
  ts: ['2014-01-01', '2014-01-02', '2014-01-03'],
  open:  [100, 101, 102],
  high:  [105, 107, 105],
  low:   [100, 101, 100],
  close: [101, 102, 103]
}

grid.moveColumnById(0, 2); // Move column 0 to position before column 2 (column index 1)
