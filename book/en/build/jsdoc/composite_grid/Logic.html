AND() → {number}

{lang=en} The AND function returns 1 (TRUE), if all arguments in List are true. It returns 0 (FALSE) if one or more arguments in List are false. Use operand && for a shorthand. {lang=ja} AND 関数は、リストのすべての引数が TRUEの 場合、1 (TRUE) を返します。リストの1つ以上の引数が FALSEの 場合、0 (FALSE) を返します。省略形は、オペランド [ && ] を使用します。 {lang=zh} 如果列表中的全部参数为真，则 AND 函数返回 1 (TRUE)。如果一个或多个列表中的参数为假，则返回 0 (FALSE)。 速记使用操作符 &&。 {lang=hant} 如果列表中的全部參數為真，則 AND 函數返回 1 (TRUE)。如果一個或多個列表中的參數為假，則返回 0 (FALSE)。速記使用操作符 &&。 {lang=ko} AND 함수는 목록의 모든 인수가 참이면 1(TRUE)을 반환합니다. 목록에 있는 하나 이상의 인수가 거짓이면 0(FALSE)을 반환합니다. 약칭으로 피연산자 &&를 사용합니다. {lang=ru} The AND function returns 1 (TRUE), if all arguments in List are true. number

{lang=en} The OR function returns 1 (TRUE), if any argument in List is true. It returns 0 (FALSE), if all arguments in List are FALSE. Use operand || for a shorthand. {lang=ja} OR 関数は、リストのいずれかの引数が TRUE の場合、1 (TRUE) を返します。リストのすべての引数が FALSE の場合、0 (FALSE) を返します。省略形は、オペランド [ || ] を使用します。 {lang=zh} 如果列表中的任意参数为真，则 OR 函数返回 1 (TRUE)。如果列表中的全部参数为假，则返回 0 (FALSE)。速记使用操作符 ||。 {lang=hant} 如果列表中的任意參數為真，則 OR 函數返回 1 (TRUE)。如果列表中的全部參數為假，則返回 0 (FALSE)。速記使用操作符 ||。 {lang=ko} OR 함수는 목록의 어떤 인수가 참이면 1(TRUE)을 반환합니다. 목록에 있는 모든 인수가 거짓이면 0(FALSE)을 반환합니다. 약칭으로 피연산자 ||를 사용합니다. {lang=ru} The OR function returns 1 (TRUE), if any argument in List is true. It returns 0 (FALSE), if all arguments in List are FALSE. Use operand || for a shorthand. {lang=de} Die ODER-Funktion gibt 1 (WAHR) aus, wenn eines der Argumente in der Liste wahr ist. Sie gibt 0 (FALSCH) aus, wenn alle Argumente in der Liste FALSCH sind. Verwenden Sie Operand || für eine Kurzschrift.

number
Example:
