stretchCell(cellRef, rowIndexopt, opt_stretchingopt, onlyToTheRightopt) → {tr.grid.Cell}

This will make the specified cell horizontally span to fit entire row regardless of position of the cell -- cell will shift to the left most of the section.
If opt_noLeftStretching is true, the cell will not extend to the left most position
This is different from cell colSpan, which the cell remain in the same position. tr-stretched class will also be added to the cell
Parameters:
cellRef
number | tr.grid.Cell
Either column index or cell reference is acceptable
rowIndex
number
<optional>
If the cell instance is given as the first parameter, this parameter will be ignored.
opt_stretching
boolean
<optional>
Default is true
onlyToTheRight
boolean
<optional>
[default: false]
By default, cell will be stretched to both left and right directions, covering entire row. Set to true to expand only to the right
Returns:
tr.grid.Cell
The cell being stretched
Example:
grid.stretchCell(1, 0); // The second cell on the first row will be stretched covering the entire row
grid.stretchCell(cellInstance, 0, true); // The given cell will be stretched covering the entire row
