tr.ElementControl()

class

new ElementControl()

Please see tr.IElementControl for full APIs and parameter lists. This is base class for all Novakit's UI Control.
All standard HTML element events are allowed and can be used through tr.IElementControl#listen
See all available event list here tr.IElementControl#event:blur
Any additional events may be provided by a derived class
