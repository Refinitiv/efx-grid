Conflator(opt_func, opt_ms, opt_thisObjopt)

class

new Conflator(opt_func, opt_ms, opt_thisObjopt)

Conflator is used to stop multiple executions of a function in the specified period of time by merging multiple calls into one single call. Conflator guarantees that only one single call will be executed.
Parameters:
opt_func
function | number
Bound function that will use this conflator
opt_ms
number | function
The delay time in millisecond before executing the function
opt_thisObj
*
<optional>
"this" object to be bound with the given function. If the function is already bound, there is no need to provide thisObj parameter
Example:
var c = new Conflator(function() { console.log("Executed"); }, 1000);
for(var i = 10; --i >= 0;) {
  c.conflate(i); // Only one "Executed" text will be logged to console after one second
}
