Type Definitions

typedef

Options

RowDefinition options
Type:
Object
Properties:
Name Type Attributes Default Description
ric string <optional>
 "" RIC for real-time fields. Prefixing 0# to the ric is equivalent to specifying asChain property.
permId string <optional>
 "" Organization PermId to be used instead of ric for private company.
values Array | Object <optional>
 null Values for static data
fields Array.<string> <optional>
 null Field that corresponds to the given static values
asChain boolean <optional>
 false The given ric will be treated as a chain
chainRic string <optional>
 "" RIC to be used for chain request (overiding ric property)
collapsed boolean <optional>
 true Chain is collapsed by default
label string | null <optional>
 null
hidden boolean <optional>
 true When this row is hidden
realTime boolean <optional>
 true Realtime row, able to request for JET/RTK
