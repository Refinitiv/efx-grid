Deleting or Removing Data

Removing rows in Atlas Blotter

Data should be removed through Atlas's blotter API, as there are resources allocated on each row such as a connection between real-time sources.

Use removeRow() or removeAllRows() from the blotter api property to add a new row.

grid.api.removeRow(0); // Remove the first row

grid.api.removeAllRows();

All available APIs for Atlas Blotter can be found here

Removing rows in Emerald Grid

Removing data is useful when you want to change a new set of data with the same Grid configuration. Removing and re-creating the entire Grid (UI + data) will always more costly than just removing the data. So, the first thing we should try is to remove the old data set and apply the new data set.

Use clearAllData() or removeAllRows() methods to clear all data.

var dt = grid.api.getDataTable();
dt.addRows([
    {"field1": "a", "field2": 1, "field3": null},
    {"field1": "b", "field2": 5, "field3": {"someKey": 10}},
    {"field1": "c", "field2": 7, "field3": [1, 2, 3]},
    {"field1": "c", "field4": "some value"}
]);
console.log(dt.getRowCount());

dt.clearAllData();
console.log(dt.getRowCount());

Use removeRow() to remove the specified row. Alternatively, you can use setRowData() with null value on the second parameter to remove the row.

var dt = grid.api.getDataTable();
var dv = grid.api.getDataView();

dt.addRows([
    {"field1": "a", "field2": 1, "field3": null},
    {"field1": "b", "field2": 5, "field3": {"someKey": 10}},
    {"field1": "c", "field2": 7, "field3": [1, 2, 3]},
    {"field1": "c", "field4": "some value"}
]);

dt.log();
var rowId = dv.getRowId(3); // the forth row

dt.removeRow(rowId);
dt.log();

dv.removeRow(2); // Remove the third row
dv.log();

dv.setRowDataAt(0, null); // Remove the first row
dv.log();

Use removeRows() to remove multiple rows at once (for better performance).
