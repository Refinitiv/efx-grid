Row Selection

The Row Selection Extension will add the ability to select a single row or multiple rows on Grid. 

efx-grid {
    height: 169px;
}

<efx-grid id="grid"></efx-grid>

import { halo } from './theme-loader.js'; // This line is only required for demo purpose. It is not relevant for your application.
await halo(); // This line is only required for demo purpose. It is not relevant for your application.

var fields = ["companyName", "market", "CF_LAST", "CF_NETCHNG", "industry"];
var records = DataGenerator.generateRecords(fields, { numRows: 15 });
var configObj = {
    sorting: {
        sortableColumns: true
    },
    columns: [
        { name: "Company", field: fields[0] },
        { name: "Market", field: fields[1], width: 120 },
        { name: "Last", field: fields[2], width: 100 },
        { name: "Net. Chng", field: fields[3], width: 100 },
        { name: "Industry", field: fields[4] }
    ],
    staticDataRows: records,
    extensions: [
        new RowSelection()
    ]
};

var grid = document.getElementById("grid");
grid.config = configObj;

basedOnContent mode

Normally, when content is sorted and a row changes position, row selection will not change position like the content does (try example above). But if you do want the row selection to relocate itself with the content, you can turn on basedOnContent mode. In this mode, row selection is made based on the content/data of the row. When clicking a row, new content/data will be set to the row and row selection will be shown according to that content/data change.

basedOnContent mode example

In the below example, try to select one or more rows, then click the column header to sort data and see what happens with row selection.

Selection modifiers and navigation

You can use Ctrl/Command + Click to select additional rows without deselecting existing row selection. Ctrl/Command + Click on the already selected row will result in deselecting the row.

Once the first row selection is made, you can use Shift + Click to select range of rows from the anchor row to the row being clicked.

You can move the row selection using Arrow keys. Use Arrow up key to move a row selection up one row. Use Arrow down key to move a row selection down one row. The Shift modifier also works with the Arrow keys.

Copying text content from selected rows

You can directly hook native copy event on Grid element to retrieve data and put it on the system clipboard. Use getSelectedRows API from the extension to get currently selected row indices. The example below illustrates how to capture copy event and store the content in TSV (Tab Separated Values) format to the system clipboard.

html hr {
    margin: 5px;
}
efx-grid {
    height: 169px;
}
textarea {
    width: 100%;
    height: 100px;
}
mark {
    opacity: 0;
    padding: 4px;
    transition: opacity 0.5s ease-out;
}

function onCopy(e) {
    var text = "No Selected Row";
    
    var selectedRowIndices = rowSelExt.getSelectedRows();
    if(selectedRowIndices) {
        var rows = grid.api.getMultipleRowData(selectedRowIndices);
        text = rows.map(recordToTSV).join("\n");
    }
    
    e.clipboardData.setData("text/plain", text);
    e.preventDefault();
    
    showKeyIndicator("Data is copied to clipboard");
}
function recordToTSV(record) {
    return ([
        record[fields[0]],
        record[fields[1]],
        record[fields[2]],
        record[fields[3]],
        record[fields[4]]
    ]).join("\t"); // TSV (Tab Separated Values)
}
function showKeyIndicator(text) {
    if(text) {
        key_indi.textContent = text;
    }
    key_indi.style.opacity = "1";
    setTimeout(onHideKeyIndicator, 1000);
}
function onHideKeyIndicator() {
    key_indi.style.opacity = "";
}

var configObj = {

        basedOnContent: true
    },
    sorting: {
        sortableColumns: true
    },
    columns: [

        { name: "Market", field: fields[1], width: 120 },

        { name: "Net. Chng", field: fields[3], width: 100 },
        { name: "Industry", field: fields[4] }
    ],
    staticDataRows: records,
    extensions: [
        rowSelExt
    ]
};

var grid = document.getElementById("grid");
grid.config = configObj;
grid.addEventListener("copy", onCopy);

Events

event

beforeSelection

Fired before selectionChanged. We can cancel the select operation by setting `cancel` property on the event arugment to true
Note that selection by JavaScript APIs cannot be prevented using this event.
Example:
let ext = new RowSelectionPlugin();
ext.listen("beforeSelection" function(e) {
  e.cancel = true; // Cancel user selection
});
event

selectionChanged

Fired when selection is changed by mouse down or key press. A change by JavaScript APIs will not trigger this event.
