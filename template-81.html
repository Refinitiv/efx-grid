Accessing Atlas Blotter data

Data in Atlas Blotter can be retrieved throughRowDefinition object.

To get row data:

var row0 = blotter.api.getRowDefinition(0); // row index 0
var row1 = blotter.api.getRowDefinition(1); // row index 1

console.log(row0.getRowData()); // No row id or index required
console.log(row1.getData("Field Name")); // No row id or index required

To set row data:

var row2 = blotter.api.getRowDefinition(2); // row index 2
var row3 = blotter.api.getRowDefinition(3); // row index 3

console.log(row2.setRowData({"Field 1": 15, "Field 2": "aa"})); // No row id or index required
console.log(row3.setData("Field Name", 77.23)); // No row id or index required

All available APIs for RowDefinition object can be found here.

For more details about accessing data, see Reading data page.
