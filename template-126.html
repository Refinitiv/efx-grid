Simple Input Formatter

This formatter creates an input without a margin and 100% width. By default the input's value will be the column's field.

Specific options

There are no specific options for this formatter.

Example

atlas-blotter {
    height: 320px;
}

<atlas-blotter id="grid"></atlas-blotter>

import './resources/extensions.js'; // This line is only required for demo purpose. It is not relevant for your application.
import { halo } from './theme-loader.js'; // This line is only required for demo purpose. It is not relevant for your application.
await halo(); // This line is only required for demo purpose. It is not relevant for your application.

/* ---------------------------------- Note ----------------------------------
  DataGenerator, Formatters and extensions are exposed to global scope
  in the bundle file to make it easier to create live examples.
  Importing formatters and extensions is still required in your application. 
  Please see the document for further information.
---------------------------------------------------------------------------*/
var fields = ["number_1", "float_1"];
var records = tr.DataGenerator.generateRecords(fields, 40);

var configObj = {
    columns: [
        { 
            name: "Column 1",
            field: fields[0],
            alignment: "center"
        },
        { 
            name: "Column 3",
            field: fields[1],
            alignment: "center"
        },
        { 
            name: "Input",
            field: fields[0],
            alignment: "center",
            binding: tr.SimpleInputFormatter.create()
        }
    ],
    staticDataRows: records
};

var grid = document.getElementById("grid");
grid.config = configObj;

Note that coral-input is now deprecated in ELF v4 (see this document for more information). It's preferably to use SimpleInputFormatter instead of CoralInputFormatter.
