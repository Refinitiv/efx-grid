Data From HTTP Request

You are free to retrieve data by any method. For instance, we can use fetch API, or XMLHttpRequest. Once you get data from the request, you can put them to Grid using APIs as described on this page.

Getting real-time data in Atlas Blotter

By default, Atlas Blotter can retrieve real-time data, if JET.Quotes2 exists in the page. To specify what data to be requested, you can define ric in row configuration and field in column configuration.

for more information about JET.Quotes2, see JET page
